# 中国空军

## 概况

抗战时期的中国空军，是以极度有限的资源对抗绝对优势敌方，
以生命为代价坚守着每一片天空。

---

## 组织架构（简）

```
空军
├── 各飞行大队（如第3大队、第4大队……）
│   ├── 各飞行中队（如8中队）
│   └── 各飞行分队
├── 中央航空学校（中央航校）
│   ├── 正式飞行员培训（各期驱逐组、轰炸组等）
│   └── 军士学校（技术与士官飞行员培训）
└── 苏联志愿航空队（援华）
```

## 飞行员培训体系

### 中央航校

- 培训周期：严苛，筛选率极高（以同期百余名学员，仅最优者获战斗机资格）
- 毕业后即分配各大队参战
- 教官群体：经验丰富，但面临"送走每一届学生、又目送他们一个个牺牲"的现实

### 牺牲与替换

| 数据 | 内容 |
|------|------|
| 平均存活时间 | 毕业至牺牲不超过6个月 |
| 平均牺牲年龄 | 不超过25岁 |
| 空军第八期总人数 | 145人 |
| 其中殉职殉国 | 46人（超过1/3） |

## 主要战役

| 时间 | 战役 | 结果 |
|------|------|------|
| 1937年8月14日 | 814笕桥空战 | 中国空军初战告捷，击落日机数架 |
| 1938年11月8日 | 成都首次空袭 | 日机18架，投弹百余枚，平民伤亡 |
| 1940年7月24日 | **成都孤胆空战** | 击落日机1架（共参战多机），中方无损失 |
| 1940年9月13日 | 重庆"九一三"空战 | 遭零式战机重创，损失惨重 |
| 1941年3月14日 | 成都空战 | 8名飞行员（含林恒）壮烈牺牲 |

## 历史评价与精神

> **"中国空军从1937年开战，凭有限的飞机扛到了1940年，付出了惨重的牺牲，但是还在坚持出击。中国没有亡，也不会亡。"**

那些驾驶着老旧双翼机、面对近千架敌机依然升空的人，  
那些在日记里只留下一句话的教官，  
那些毕业不到半年便血洒长空的学生，  

共同构成了中国空军在这段岁月中最真实的样貌。
