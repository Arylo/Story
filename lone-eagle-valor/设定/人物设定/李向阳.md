# 李向阳

## 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 李向阳 |
| 身份 | 中央航空学校飞行教官 |
| 任教期数 | 含空军第八期（145人，殉国46人） |
| 结局 | 战争结束后已去世，死亡时间不详 |
| 史料来源 | 其日记、旅日作家萨苏考证、《航空知识》2015年8月号 |

## 核心特质

- 不是传统意义上的"英雄主义"展示——他不是要漂亮地亮相
- 他带着对已殉国学生极深的悲痛与伤感升空
- 他是替学生去报仇，是跟学生们一起死，去拼命的
- 日记中对那场空战只字未提，只有一句话留下了他的全部心情

## 日记原文

> **"我的学生都战死了，现在该我这个老师上去了。"**

这十七个字，是整个故事的情感核心。
