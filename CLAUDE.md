# Story 创作仓库说明

本仓库包含多个独立故事，每个顶层文件夹对应一个故事项目。

## 故事列表

| 文件夹 | 英文名 | 中文名 |
|--------|--------|--------|
| `all-are-heroes/` | All are Heroes | 人人皆为英雄 |
| `lone-eagle-valor/` | Lone Eagle's Valor | 长空孤胆 |

## 目录结构（每个故事通用）

```
<故事文件夹>/
├── 时间线/      # 按年份记录事件（如 2017.md）
├── 故事/
│   ├── 正文/      # 正式章节（按序号命名，如 01-01.md）
│   └── 大纲/      # 章节大纲与剧情规划
└── 设定/
    ├── 世界观/    # 背景、体系、国家等设定文档
    └── 角色设计/  # 角色档案与角色索引
```

## 约定

- 每个故事文件夹相互独立，设定不共享
- 故事正文按 `章节-小节` 格式命名（如 `01-01.md`）
- 时间线文件以年份命名（如 `2017.md`），每个文件记录该年发生的事件与变化
- 子文件夹内有独立的 `CLAUDE.md` / `AGENT.md` 说明文件
