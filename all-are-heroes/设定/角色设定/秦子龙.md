# 秦子龙

## 基本信息

| 属性 | 内容 |
| ---- | ---- |
| 出身 | [华夏](../世界观/国家-华夏.md) |
| 性别 / 年龄 | 男 / 年龄待定 |
| 明面身份 | 初入炼气的小透明 |
| 转折地点 | 偶遇大能冲阶时的雷劫现场 |
| 共生对象 | [沈华强](沈华强.md) 残魂 |
| 对应体系 | [炼气体系](../世界观/修炼体系-炼气.md) 为主，兼具 [炼体体系](../世界观/修炼体系-炼体.md) 见识 |
| 核心词条 | 祖传玉佩、夺舍未遂、灵魂共生 |

## 核心设定

- 他原本只是华夏修行体系里一个不起眼的炼气新人，资质、背景和名望都不出众。
- 某次外出途中，他意外撞见一位炼体大能冲击更高境界时引来的雷劫现场。
- 那位名为 [沈华强](沈华强.md) 的大能由于准备仓促，升阶失败，肉身受毁，只能灵魂出窍，打算另寻躯壳重头开始。
- 在逃离与寻体的过程中，对方刚好遇上秦子龙。
- 为了走捷径，而不是重新做一个婴孩或慢慢布局，沈华强直接选择强行夺舍秦子龙的身体。
- 这场夺舍之所以失败，不是单纯靠运气，而是因为他随身佩戴的祖传护身玉佩在关键时刻护住了神魂。
- 玉佩没能彻底磨灭对方残魂，却成功阻止了灵魂覆盖，最终逼成了共生局面。
- 从那之后，他体内便住进了一个本想直接拿走他人生的老怪物。

## 开局状态

- 身体主导权依旧在秦子龙手里，但 [沈华强](沈华强.md) 的残魂并没有真正离开。
- 那个升阶失败的大能既失去了原本肉身，也失去了重新高高在上的资格，只能暂时寄存在他体内。
- 在危险、修炼瓶颈或某些特殊状况下，残魂能够提供经验、判断，甚至短暂接管级别的帮助。
- 那块祖传护身玉佩也因此从普通家传之物，变成了他身上最关键的秘密之一。

## 核心矛盾

- 他想活成自己，不想成为沈华强留下来的备用容器。
- 沈华强既看不上这个小人物，又不得不承认自己现在只能依附他存在。
- 玉佩护住了他一次，却也意味着他之后可能会被更多人盯上，因为这件东西本身就不简单。

## 叙事作用

- 负责打开华夏线、炼气与炼体双线冲突，以及“弱者如何承载强者残魂”的成长主题。
- 也是六人中最适合展开师徒感、寄生感、宿敌感三重关系张力的角色。

## 成长方向

- 前期仍以炼气成长为主，但会不断从沈华强处偷学炼体知识。
- 中期可发展成罕见的气体双修胚子，也可逐步揭开祖传玉佩本身的来历。
- 若后续需要强化冲突，可以让玉佩成为双方争夺身体主导权之外的第二个关键点。
