# 藤原智恵

## 基本信息

| 属性 | 内容 |
| ---- | ---- |
| 身份 | 藤原新之助之母 |
| 姓名 | 藤原智恵（原名：朴智恵） |
| 明面身份 | 棒之日普通家庭中的温和母亲 |
| 真正来历 | [釜门基地市](../世界观/基地市-釜门.md) 原棒之国旧豪门之女 |
| 出生年份 | 灭妖历229年 |
| 血脉背景 | 家族中曾多次出现大地觉醒者 |
| 当前状态 | 与恋人私奔离家后，低调定居于 [对马基地市](../世界观/基地市-对马.md) |

## 核心设定

- 她外表与日常表现都极为普通，很容易被误认为只是平凡母亲。
- 实际上，她出身于 [釜门基地市](../世界观/基地市-釜门.md) 原棒之国旧豪门（朴氏），家族内部保留了强烈的阶层观与精英式家教，其家系中也曾多次出现 **大地** 觉醒者。
- 她与出身 [户门基地市](../世界观/基地市-户门.md) 古老家族的藤原道长并非循规蹈矩地结合，而是直接私奔离家，最终在位于两城之间的 [对马基地市](../世界观/基地市-对马.md) 隐姓埋名。
- 藤原新之助出生于灭妖历256年的 [对马基地市](../世界观/基地市-对马.md)，那一年她27岁。
- 她身上携带的，不只是旧家族出身，还有可能影响藤原新之助属性异常与血统异常的关键线索。

## 剧情价值

- 她能为藤原新之助的母系血脉、原棒之国旧豪门（朴氏）遗产与 **大地** 血统来源提供另一半解释。
- 她与藤原道长的结合，本身就像是 [釜门基地市](../世界观/基地市-釜门.md)（朴氏） 与 [户门基地市](../世界观/基地市-户门.md)（藤原氏） 两条旧血脉线在 [对马基地市](../世界观/基地市-对马.md) 完成成功的隐秘重叠。
