# 祈礼

## 基本信息

| 属性 | 内容 |
| ---- | ---- |
| 落点 | [铁熊](../世界观/国家-铁熊.md) |
| 性别 / 年龄 | 男 / 30岁 |
| 原世界身份 | 来自异世界的母胎单身社畜 |
| 身体状态 | 长期 996 导致偏孱弱、亚健康 |
| 降临方式 | 因神明失误死亡后，被补偿性肉身投放 |
| 对应体系 | [魔法体系](../世界观/修炼体系-魔法.md) |
| 核心词条 | 肉身穿越、神明补偿 |

## 核心设定

- 他原本只是异世界里普普通通的三十岁社畜，因为神明失误而意外死亡。
- 长年 996 的工作方式让他身体偏孱弱、长期亚健康，属于那种一眼就不像能打的人。
- 作为补偿，神明满足了他“想成为能释放魔法的魔法师”的愿望。
- 与常见的灵魂穿越不同，他是整个人连同原本肉身一起被投放到这个世界，更接近 [人物词条](../世界观/人物词条.md) 中的肉身穿越。
- 他的落点偏偏是高压、强国家、重军工的 [铁熊](../世界观/国家-铁熊.md)，一个最不适合普通外来者慢慢适应的魔法国度。

## 开局状态

- 他终于来到了真正存在魔法的世界，却没有身份、没有背景、没有本地常识。
- 更糟的是，他落地时带来的还是原本那副被 996 榨得偏弱、亚健康的身体，与铁熊那种普遍高大厚重的北地体格形成鲜明反差。
- 神明补偿只保证他“有机会成为魔法师”，并没有直接把力量塞进他手里。
- 所以他的开局并不是爽文式一步登天，而是一个现代普通人突然掉进超凡战争社会后的硬着陆。

## 核心矛盾

- 愿望实现了，但他必须先证明自己有资格活进这个愿望里。
- 他既羡慕魔法世界，又必须承受这个世界远比幻想更冷酷、更制度化的一面。
- 在一个连普通男性都往往是大块肌肉的北地社会里，他那副亚健康社畜身体会让他在最初阶段格外格格不入。
- 神明的补偿究竟只是一次投放，还是暗中留下了某种门路，也是后续悬念之一。

## 叙事作用

- 负责打开铁熊线、魔法体系线，并承担“现代普通人面对超凡军工国家”的外来者视角。
- 他是六人里最典型的读者代入入口之一，负责把陌生设定转成直观生存问题。

## 成长方向

- 前期适合走学院、军方附属或边疆法师团路线，以求先获得合法身份和施法资格。
- 中后期可以围绕异界知识、魔法适性与神明补偿是否留后门继续展开。
