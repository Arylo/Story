# 史龙

## 基本信息

| 属性 | 内容 |
| ---- | ---- |
| 身份定位 | 灭妖历元年的传奇人物 |
| 性别 / 年龄 | 男 / 22岁（灭妖历元年时） |
| 出身 | 华夏 |
| 对应体系 | 炼体体系 |
| 核心词条 | 炼体奇才、天生战斗狂、直播斩妖 |
| 叙事定位 | 世界观背景英雄，非当前六人主角群成员 |

## 核心设定

- 他是妖兽潮初期最早被全世界记住名字的人类强者之一。
- 史龙天生就适合炼体一道，肉身底子、战斗直觉与临场厮杀欲都远超常人，是标准意义上的炼体奇才。
- 与其说他热爱胜负，不如说他天生就为搏杀而生，越是逼近生死线，越能打出超越常理的凶悍状态。
- 灭妖历元年，年仅二十二岁的他在一场近乎疯狂的直播中，孤身闯入高阶妖兽巢穴，最终凭一己之力斩杀高阶妖兽。
- 这场直播成为旧时代秩序崩塌后最关键的转折点之一，让更多人第一次相信，人类并非只能依赖失效中的旧时代火力。

## 历史影响

- 他的胜利直接点燃了人类反击妖兽的信心，也让旧时代武道与炼体传承重新被大规模重视。
- 对后世的炼体修行者而言，史龙既是榜样，也是某种近乎神话的标尺。
- 在不少幸存者叙事里，他代表的不只是“强”，更是灾变初期人类第一次真正反扑的胆魄。

## 叙事作用

- 适合作为世界观背景中的标志性人物，用来解释灭妖历元年的精神起点与炼体体系的早期声望来源。
- 也可以成为后世角色口中的传奇参照，让读者更直观理解“高阶炼体者”究竟意味着什么。
