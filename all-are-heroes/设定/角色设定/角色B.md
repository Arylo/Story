# 角色B

| 属性 | 内容 |
| ---- | ---- |
| 出身 | [横鹰](../world-buildings/国家-横鹰.md) |

| 明面身份 | 横鹰某批次改造计划中的普通参与者 |

| 对应体系 | [改造体系](../world-buildings/修炼体系-改造.md) |
| 核心词条 | F级狼人化异能、真祖复苏、吸血升阶 |

## 核心设定

- 她原本只是众多接受改造手术者中的一员，起点并不特殊，也不被任何人重点关注。
- 术后稳定下来后，她觉醒出的本命能力不是惊艳全场的高阶异能，而是最低级的 F 级狼人化异能。
- 这意味着她在横鹰标准化评价体系中，属于最不起眼、最不被看好的底层改造者之一。
- 然而在这具被判定为低潜力的狼化改造体内，苏醒的却像是某位沉睡千年的吸血鬼真祖意识。
- 由于是真祖级存在，她实际上还保留了部分吸血鬼真祖的残留能力。
- 其中一项最关键的残能，是**吸取纯人类鲜血后，可以提升能量炉等级和异能等级**，这本质上属于极端异常的变异式成长。
- 她自己也无法立刻分清，这究竟是古老存在复苏、灵魂错位，还是更高层次的异常回归。

## 开局状态

- 在外界眼里，她只是“这批人里又一个 F 级狼人化失败品边缘体”，并不值得投入太多资源。
- 这种低评价反而构成保护色，使她暂时没有被横鹰中枢立刻视作战略资源。
- 可她本人并不知道，自己其实还藏着能够靠吸血提升炉心与异能的恐怖上限。
- 偏偏横鹰纯人类本就稀少，绝大多数人都已接受改造手术，再加上她根本没有获取受控血液样本的渠道，因此这个能力长期处于完全未知状态。
- 真祖意识逐步复苏后，她开始在低劣狼化本能与古老高位掠食者意识之间产生严重撕裂。

## 核心矛盾

- 她的公开评级极低，但体内潜藏的东西极可能远远超出横鹰现有体系的理解上限。
- F 级狼人化异能既是枷锁，也是遮蔽物，让她能活下来，却也让她难以被认真对待。
- 她真正危险的地方不是眼下的低级狼人化，而是那条连她自己都没意识到的“吸食纯人类鲜血即可跳级成长”的暗线。
- 一旦真祖特征被外界确认，她的身份会立刻从“普通底层改造者”变成“必须控制的危险异常”。

## 叙事作用

- 负责打开横鹰线、改造体系线，以及“人工改造体系能否承载远古超凡本质”的实验性谜团。
- 她的戏剧点不在高开高走，而在最不起眼的样本中潜伏着最不可控的东西。

## 成长方向

- 前期以 F 级狼人化的低配现实求生，强调她在制度中被低估、被忽略，也强调她对自己真正的成长机制一无所知。
- 中期逐渐挖出真祖记忆，让狼性、血族性与改造体系发生冲突、融合或反噬，并慢慢逼近“鲜血可以让她升级”的真相。
- 后期可以把“最低级的外壳”与“最高位的苏醒者”做成她最强的反差标签。
