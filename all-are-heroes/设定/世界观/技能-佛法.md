# 佛法

佛法是佛魔体系修行者通过**念诵经文**，引动**愿力**或**业力**而施展的特殊术法体系。顺向修行者通常以愿力为根基，逆向修行者通常以业力为根基，二者都极度依赖心性、经文理解与自身承载能力。

## 力量来源

- **愿力**：多由顺向修行者积累，常用于守护、镇压、净化、加持等方向。
- **业力**：多由逆向修行者积累，常用于侵蚀、压制、反噬、诡变等方向。
- 无论是**愿力**还是**业力**，都可以用来增强佛法的**威力、范围与距离**。
- 施法者可以借用其他修炼者的**愿力**或**业力**来施法，但前提是自身**能够承受**；若超出承载极限，便可能引发反噬、失控或心神受损。

## 品阶等级（高 → 低）

| 品阶 | 说明 |
|------|------|
| 本愿法 | 最高层次的佛法，直指修行者本愿与根本道途，往往涉及大范围镇压、超强净化或逆转局势 |
| 佛光法 | 仅次于本愿法，光明浩大，常见于高层传承与大型护持、净化类经术 |
| 圣法 | 威力强大，结构成熟，已具备极高实战与镇压价值 |
| 贤法 | 威力中等，已有成熟战斗与辅助价值，是多数佛魔修士的中坚层级 |
| 凡法 | 最基础的入门佛法，结构简单，是初学者接触经术的起点 |

## 释放方式

- 佛法的核心释放方式是**念诵经文**。
- 经文越完整，理解越深，引动的愿力或业力通常越稳定。
- 借用他人愿力或业力时，也必须通过经文完成统合与导出，否则极易失控。

## 使用者

- 主要由**佛魔体系**修行者修炼和使用。
- **顺向**修行者多以愿力施展佛法。
- **逆向**修行者多以业力施展佛法。

## 与体技、灵技、魔法、活气的区别

- **佛法**：依赖念诵经文与愿力/业力引动，可借他人之力增强威力、范围与距离。
- **体技**：依赖肉身与气血，战斗方式最直接粗暴。
- **灵技**：依赖灵气修为与法诀，偏向术法演化与灵气操控。
- **魔法**：依赖魔力、术式结构与属性亲和，释放方式最复杂。
- **活气**：依赖活气与外物媒介，强调属性附着、媒介强化与物件联动。
