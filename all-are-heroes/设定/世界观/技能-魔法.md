# 魔法

魔法是通过调动魔力、构建术式与触发元素或法则反应来完成的超凡能力，是魔法体系修行者最核心的战斗、辅助与研究手段。

## 属性分类

| 属性 | 说明 |
|------|------|
| 无 | 无明显元素倾向，常用于护盾、增幅、塑形与通用术式 |
| 风 | 速度快，擅长切割、位移与范围扰动 |
| 雷 | 爆发力强，擅长穿透、麻痹与瞬间杀伤 |
| 水 | 流动多变，擅长治愈、束缚与持续消耗 |
| 火 | 破坏性强，擅长灼烧、爆裂与范围打击 |
| 土 | 稳重厚实，擅长防御、加固与地形控制 |
| 冰 | 由水属性变异而来，擅长冻结、减速与封锁行动 |
| 光 | 擅长净化、治疗、守护与增益 |
| 暗 | 擅长隐匿、侵蚀、诅咒与削弱 |
| 时间 | 最为稀有，擅长迟缓、加速与短暂干涉时间流 |
| 空间 | 仅次于时间，擅长传送、切割、储物与结界 |

## 属性亲和

- 一般人通常只亲和 **一种属性**。
- 多属性亲和者极少，往往会被视为天才或异类。
- **时间** 亲和最为稀有，**空间** 亲和次之。
- **冰属性** 属于水属性的变异属性，存在冰属性魔法，但不存在天然的冰属性亲和。
- 非亲和属性者也可以释放对应属性的魔法，但威力、稳定性与操控精度通常不及亲和属性者。

## 复合魔法

- 魔法可以由多个属性进行合成，形成复合魔法。
- 复合魔法往往结构更复杂、耗魔更高，但威力、范围与变化也会更强。
- 属性之间的相性不同，合成后的效果也会随组合不同而发生巨大差异。

常见例子：

- **风 + 火**：形成火风暴、爆炎流等高扩散术式。
- **雷 + 火**：形成高爆发的灼雷术式。
- **土 + 光**：形成防御与净化并存的圣壁类术式。
- **空间 + 雷**：形成高机动、突袭型复合魔法。

## 品级等级（低 → 高）

| 品级 | 说明 |
|------|------|
| 下级 | 最基础的民用或入门战斗魔法，结构简单，耗魔较低 |
| 中级 | 已具备较稳定的实战价值，是普通施法者常用的核心层级 |
| 上级 | 威力明显增强，对魔力控制与咏唱要求更高 |
| 最上位 | 超越常规上级魔法，术式复杂，足以改变局部战局 |
| 圣级 | 稀有而强大，往往涉及大范围破坏、净化或守护 |
| 王级 | 足以影响一城一域的高等魔法，施展门槛极高 |
| 帝级 | 近乎战略级的恐怖术式，往往只掌握在顶尖强者手中 |
| 神级 | 传说中的终极魔法，足以干涉天象、空间或法则 |

## 释放方式

| 方式 | 说明 |
|------|------|
| 咏唱 | 最常见的施法方式，通过言灵引导术式成型 |
| 完全咏唱 | 将咒文完整释放，施法最稳定，威力通常最高 |
| 高速咏唱 | 压缩咏唱时间，提升施法速度，但控制难度更高 |
| 二重咏唱 | 同时维持两道咏唱结构，形成双重术式 |
| 三重咏唱 | 比二重咏唱更复杂，可叠加多道术式，施法负担极大 |
| 无咏唱 | 直接以精神力与术式理解发动魔法，速度最快，门槛极高 |
| 演奏 | 借助乐器、旋律或节奏作为媒介施法，常见于特殊流派 |
| 魔法阵 | 通过预设术式回路施法，适合大型、群体或持续型魔法 |
| 手势结印 | 以手势快速引导术式，强调速度与精度 |
| 符文雕刻 | 将术式固化在材料、器具或地面上，延迟触发或长期生效 |
| 组合咏唱 | 将咏唱与结印结合，兼顾稳定性与速度 |
| 光环被动 | 以自身为核心持续释放效果，不依赖瞬时爆发而依赖长时间维持 |

## 施法者

魔法体系修行者自 **魔法学徒** 起开始接触魔力，自 **见习魔法师** 起进一步掌控体内魔力，自 **魔法师** 起则能较成熟地施展各类成体系魔法。

## 与灵技、体技、活气、佛法的区别

- **魔法**：依赖魔力、术式结构与元素亲和，变化最复杂，远中程优势明显。
- **灵技**：依赖灵气修为与术法传承，重在灵气驾驭与法术演化。
- **体技**：依赖肉身、气血与近战爆发，战斗方式最直接粗暴。
- **活气**：依赖活气与外物媒介，强调属性附着、媒介强化与物件联动。
- **佛法**：依赖念诵经文与愿力/业力，能借他人之力放大威力，但整体不以元素亲和与术式结构为核心。
