# 修炼体系总览

当前世界的修炼道路主要分为七条主线：**炼气体系**、**炼体体系**、**骑士道体系**、**魔法体系**、**活气体系**、**佛魔体系** 与 **改造体系**。

- **炼气体系**：偏向灵气、法术、灵技与灵根资质。
- **炼体体系**：偏向肉身、气血、体技与近身搏杀。
- **骑士道体系**：偏向骑士誓约、圣暗双属性统合，以及体技正面突破。
- **魔法体系**：偏向魔力、术式、元素法则与精神掌控。
- **活气体系**：偏向属性觉醒、外物附着与媒介强化。
- **佛魔体系**：偏向心性、愿力、业力与正逆观照。
- **改造体系**：偏向外科改造、能量炉、稳定供能与异能驱动。

七条体系都以 **普通人** 为起点。为便于总览对比，下文统一以 **普通人** 指代各体系尚未踏上修炼道路时的初始状态。其中炼气体系、炼体体系与骑士道体系共享 **锻体** 这一入门阶段，骑士道会在锻体之后先进入 **骑士侍从** 阶段感应未来属性，再于 **见习白骑士**、**见习黑骑士** 与 **见习灰骑士** 三脉中完成分流，后期分别走向 **神圣**、**黑暗** 与 **混沌** 三条高阶路线。灰系在 **灰骑士** 阶段通常弱于同境白 / 黑系，但一旦迈入 **混沌阶段** 并完成圣暗双属性统合，往往会反过来优于同境 **神圣 / 黑暗系**；魔法体系自 **魔法学徒** 起开始感知和吸收魔力，活气体系则需先完成觉醒后才能进入 **一阶**，佛魔体系会在立信之后分化为 **顺向** 与 **逆向**，分别自 **净信** 或 **染信** 起步，而改造体系则需先通过手术将 **能量炉** 代替心脏，并在炉心运转满 **一天** 后确定等级，再进入 **一级改造者**。活气体系的觉醒主要分为 **自行觉醒** 与 **基因药剂觉醒** 两类，其中触摸觉醒石觉醒也属于自行觉醒，因此自行觉醒只能在 10 岁至 20 岁之间完成，基因药剂觉醒则在 14 岁后即可尝试。

## 一、七大修炼主线

总体修炼路线：

```text
普通人
     ├→ 锻体【炼气、炼体与骑士道共同起点，可接触体技】
     │    ├→ 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 合体 → 大乘 → 渡劫
     │    ├→ 武徒 → 武师 → 武王 → 武皇 → 武帝 → 武尊 → 武圣 → 武神
     │    └→ 骑士侍从 → 见习白/黑/灰骑士 → 白/黑/灰骑士 → 精锐白/黑/灰骑士 → 神圣/黑暗/混沌骑士 → 神圣/黑暗/混沌领主 → 神圣/黑暗/混沌帝王
     ├→ 魔法学徒 → 见习魔法师 → 魔法师 → 魔导士 → 魔导师 → 大魔导师 → 法神 → 贤者 → 大贤者
     ├→ 一阶 → 二阶 → 三阶 → 四阶 → 五阶 → 六阶 → 七阶 → 八阶 → 九阶
     ├→ 净信 → 念住 → 明心 → 见道 → 离欲 → 无诤 → 不动 → 补处 → 正觉
     ├→ 染信 → 逆观 → 破执 → 逆行 → 同体 → 妄语 → 大动 → 补处 → 正觉
     └→ 一级改造者 → 二级改造者 → 三级改造者 → 四级改造者 → 五级改造者 → 六级改造者 → 七级改造者 → 八级改造者 → 九级改造者
```

### 1. 炼气体系

分支路线：

```text
普通人
  └→ 锻体
      └→ 炼气
           └→ 筑基
                └→ 金丹
                     └→ 元婴
                          └→ 化神
                               └→ 合体
                                    └→ 大乘
                                         └→ 渡劫
```

特点：

- 每个大阶段分为 **一重至九重** 加 **圆满**，共 **10 个小境界**。
- 除普通人外，各阶段都可使用 **体技**，但在**整个炼气体系中，体技最高只能修炼到小成**。
- 自 **炼气期** 起开始驾驭灵气，并优先修炼、使用 **灵技**。
- **筑基期** 可以借助法器或飞行宝具飞行。
- **元婴期** 可以无需借物，直接御空飞行。
- **渡劫期** 为成仙前最后关口，需要承受天劫。

境界统计：

- **9 个大阶段 × 10 个小境界 = 90 个修炼境界层次**。

详见：[修炼体系-炼气.md](修炼体系-炼气.md)

### 2. 炼体体系

分支路线：

```text
普通人
  └→ 锻体
      └→ 武徒
           └→ 武师
                └→ 武王
                     └→ 武皇
                          └→ 武帝
                               └→ 武尊
                                    └→ 武圣
                                         └→ 武神
```

特点：

- 每个大阶段分为 **一重至五重** 加 **圆满**，共 **6 个小境界**。
- 自 **锻体阶段** 起开始掌握 **体技**，且在锻体阶段体技最高只能达到 **小成**。
- 进入 **武徒** 后，体内开始诞生 **气血之力**。
- **气血之力** 可附着在体技之上，增强威力与攻击范围。
- 进入 **武皇** 后，可以凭借 **气血之力御空飞行**。
- 炼体体系会持续深修体技，一路延伸到武神。

境界统计：

- **9 个大阶段 × 6 个小境界 = 54 个修炼境界层次**。

详见：[修炼体系-炼体.md](修炼体系-炼体.md)

### 3. 骑士道体系

分支路线：

```text
普通人
  └→ 锻体
      └→ 骑士侍从
           ├→ 见习白骑士
           │    └→ 白骑士
           │         └→ 精锐白骑士
           │              └→ 神圣骑士
           │                   └→ 神圣领主
           │                        └→ 神圣帝王
           ├→ 见习黑骑士
           │    └→ 黑骑士
           │         └→ 精锐黑骑士
           │              └→ 黑暗骑士
           │                   └→ 黑暗领主
           │                        └→ 黑暗帝王
           └→ 见习灰骑士
                └→ 灰骑士
                     └→ 精锐灰骑士
                          └→ 混沌骑士
                               └→ 混沌领主
                                    └→ 混沌帝王
```

特点：

- 骑士道每个大阶段分为 **一重至五重** 加 **圆满**，共 **6 个小境界**。
- 修行者在 **锻体圆满** 后，必须先进入 **骑士侍从** 阶段，开始感应未来对应属性，但此时仍不能将属性附加在体技上。
- 进入 **见习白骑士**、**见习黑骑士** 或 **见习灰骑士** 后，体内已经能**自行产生对应属性**。
- 自 **白/黑/灰骑士** 起，可从周围吸收同类属性并注入体技：白为 **圣属性**，黑为 **暗属性**，灰与混沌系则始终为 **圣暗双属性**。
- 自 **骑士** 阶段起，可以凝聚 **一对光翼** 短暂御空飞行；至 **精锐骑士** 阶段，拥有 **两对光翼**，可稳定御空飞行；此后每升一阶再增 **一对光翼**，直到帝王达到 **五对光翼**。
- **灰骑士** 与 **精锐灰骑士** 阶段通常弱于同境白 / 黑系，但一旦迈入 **混沌阶段**，便会反过来优于同境 **神圣 / 黑暗系**。
- 骑士道体技内部称谓为 **大帝级、国王级、爵士级、凡级**，实则分别对应常规体技中的 **天、地、玄、黄**。

境界统计：

- **8 个大阶段 × 6 个小境界 = 48 个修炼境界层次**。

详见：[修炼体系-骑士道.md](修炼体系-骑士道.md)

### 4. 魔法体系

分支路线：

```text
普通人
  └→ 魔法学徒
       └→ 见习魔法师
            └→ 魔法师
                 └→ 魔导士
                      └→ 魔导师
                           └→ 大魔导师
                                └→ 法神
                                     └→ 贤者
                                          └→ 大贤者
```

特点：

- 每个大阶段分为 **初阶、中阶、高阶**，共 **3 个小境界**。
- **魔法学徒** 开始从周围吸收游离魔力。
- **见习魔法师** 除了继续从周围吸收魔力，还会在体内开始诞生魔力。
- **魔法师** 可以借助法杖、魔导器或飞行道具飞行。
- **魔导士** 可以短暂御空飞行。
- **魔导师** 可以稳定御空飞行。
- **大魔导师** 可以携带其他人一起御空飞行。
- 魔法师拥有元素亲和体质，常见亲和包括 **风、雷、水、火、土、光、暗、时间、空间**。
- 魔法本身则可分为 **无、风、雷、水、火、土、冰、光、暗、时间、空间** 等属性，其中 **冰属性** 属于水属性的变异属性，不对应天然亲和属性。
- 一般人通常只亲和 **一种属性**，其中 **时间** 最为稀有，**空间** 次之。
- 魔法体系内部通常将不会魔法的人称为 **麻瓜**。

境界统计：

- **9 个大阶段 × 3 个小境界 = 27 个修炼境界层次**。

详见：[修炼体系-魔法.md](修炼体系-魔法.md)

### 5. 活气体系

分支路线：

```text
普通人
  └→ 一阶
       └→ 二阶
            └→ 三阶
                 └→ 四阶
                      └→ 五阶
                           └→ 六阶
                                └→ 七阶
                                     └→ 八阶
                                          └→ 九阶
```

特点：

- 每个大阶段分为 **初期、中期、后期、巅峰**，共 **4 个小境界**。
- 活气的觉醒主要分为 **自行觉醒** 与 **基因药剂觉醒** 两类，其中**触摸觉醒石觉醒也属于自行觉醒**，因此自行觉醒只能在 **10 岁至 20 岁** 之间完成，而**基因药剂觉醒**只要 **14 岁后** 即可尝试。
- 觉醒属性分为 **大空七属性** 与 **大地七属性**，其中 **大空** 与 **大地** 为尊属性。
- 通过**自行觉醒**，包括自然自行觉醒与触摸觉醒石觉醒，都有概率觉醒 **大空** 或 **大地** 属性。
- 正常情况下，一名活气修行者只会拥有**一种**活气属性，双属性同体近乎不可能，更别说 **大空** 与 **大地** 同体。
- 使用基因药剂觉醒者只能觉醒 **非尊属性**，且基因药剂觉醒存在失败概率，一生最多只能尝试 **三次**。
- 使用基因药剂觉醒者后续进阶时，也只能使用**下一阶基因药剂**。
- 达到 **一阶** 时，体内开始产生活气，并能使用活气技能。
- 达到 **二阶** 时，**大空** 与 **大地** 属性可借物飞行。
- 达到 **三阶** 时，其余属性也可借物飞行。
- 达到 **四阶** 时，**大空** 与 **大地** 属性可御空飞行。
- 达到 **五阶** 时，其余属性也可御空飞行。
- 觉醒属性详见：[技能-活气.md](技能-活气.md)

境界统计：

- **9 个大阶段 × 4 个小境界 = 36 个修炼境界层次**。

详见：[修炼体系-活气.md](修炼体系-活气.md)

### 6. 佛魔体系

分支路线：

```text
普通人
  ├→ 净信
  │    └→ 念住
  │         └→ 明心
  │              └→ 见道
  │                   └→ 离欲
  │                        └→ 无诤
  │                             └→ 不动
  │                                  └→ 补处
  │                                       └→ 正觉
  └→ 染信
       └→ 逆观
            └→ 破执
                 └→ 逆行
                      └→ 同体
                           └→ 妄语
                                └→ 大动
                                     └→ 补处
                                          └→ 正觉
```

特点：

- 佛魔体系分为 **顺向** 与 **逆向** 两种修炼方向。
- 顺向大阶段依次为 **净信、念住、明心、见道、离欲、无诤、不动、补处、正觉**。
- 逆向大阶段依次为 **染信、逆观、破执、逆行、同体、妄语、大动、补处、正觉**。
- 顺向与逆向的每个大阶段，都分为 **一重至八重** 加 **圆满**，共 **9 个小境界**。
- **净信** 时可产生 **愿力**，**染信** 时可产生 **业力**。
- **念住** 时可吸收 **愿力**，**逆观** 时可吸收 **业力**。
- 顺向修行更重 **清净、观照、离欲与不动**。
- 逆向修行更重 **逆观、破执、逆行与以魔炼心**。
- **见道** 与 **逆行** 可短暂御空飞行。
- **离欲** 与 **同体** 可御空飞行。
- **不动** 与 **大动** 可显现法相。
- 两条道路虽然过程截然不同，但最终都以 **补处** 与 **正觉** 为终点。

境界统计：

- **顺向 9 个大阶段 × 9 个小境界 = 81 个修炼境界层次**。
- **逆向 9 个大阶段 × 9 个小境界 = 81 个修炼境界层次**。

详见：[修炼体系-佛魔.md](修炼体系-佛魔.md)

### 7. 改造体系

分支路线：

```text
普通人
  └→ 一级改造者
       └→ 二级改造者
            └→ 三级改造者
                 └→ 四级改造者
                      └→ 五级改造者
                           └→ 六级改造者
                                └→ 七级改造者
                                     └→ 八级改造者
                                          └→ 九级改造者
```

特点：

- 进入改造体系的前提，是通过**外科手术**将 **能量炉** 代替心脏。
- 能量炉必须在体内连续运转 **一天** 后，才能确定自身等级。
- **一人一生只能激活一个能量炉**，且**每个能量炉只能绑定一个人**。
- 能量炉分为 **S、A、B、C、D、E、F** 七个等级，一般情况下不会升级，只有发生**变异**时才可能提升。
- 每个能量炉都会自带一个与自身等级相同的**本命异能**。
- 每次能量炉完成一次大阶段升级，都会新增一个与本命异能相关的异能能力；新增异能可通过 **异能水晶** 进行替换。
- 改造体系的大阶段固定为 **一级** 到 **九级**。
- 与其他体系不同，改造体系不设固定小境界，而是以每个大阶段内部 **0% 至 100%** 的能量充能进度作为成长刻度；只有充能达到 **100%** 时，才能晋升下一阶段。
- 自 **五级改造者** 起，通常可凭借充盈的炉能输出 **御空飞行**。
- 若 **本命异能** 或 **衍生异能** 本身具备飞行、浮空、反重力等特殊能力，则也可能在 **一级阶段** 便提前获得飞行能力。
- 异能规则详见：[技能-异能.md](技能-异能.md)

境界统计：

- **9 个大阶段**，每个大阶段内部统一按 **0% 至 100% 充能进度**计算，不另设固定小境界。

详见：[修炼体系-改造.md](修炼体系-改造.md)

## 二、基地市镇守者

灾变时代里，基地市能否长期存在，从来不只取决于城墙、工业与人口，更取决于是否有足以镇住一城命脉的顶级强者。如今各国的共同底线，是**每座基地市至少常驻一位九阶镇守者**。这名镇守者未必是城中唯一最强者，但必须足以在城防崩溃前扛住最坏局面。

不同体系对应的九阶称谓通常如下：

- **炼气体系**：渡劫期
- **炼体体系**：武神
- **骑士道体系**：帝王级骑士（神圣帝王、黑暗帝王或混沌帝王）
- **魔法体系**：大贤者
- **活气体系**：九阶活气强者
- **佛魔体系**：正觉
- **改造体系**：九级改造者

多数国家在和平时只维持“至少一位”这一底线，战时则会额外增派同级强者驻城、轮值或护送战略物资。

## 三、体技、灵技、魔法、活气、佛法与异能

### 1. 体技

适用范围：

- 普通人无法使用。
- 炼气体系、炼体体系与骑士道体系的修士，自 **锻体** 起都可使用体技。
- 炼气体系中，体技只是辅修手段，且**上限始终为小成**。
- 炼体体系中，体技是核心战斗方式，可随境界持续精进。
- 骑士道体系中，体技同样是核心战斗方式；自 **骑士侍从** 起开始感应属性，**见习骑士** 阶段可自行产生属性，正式骑士起可吸收外界同类属性并注入体技，灰 / 混沌系则始终以 **圣暗双属性** 为根本，只是前期偏弱、后期统合后更强。

等级结构：

- 品阶：**天、地、玄、黄**
- 骑士道别称：**大帝级、国王级、爵士级、凡级**（对应 **天、地、玄、黄**）
- 熟练度：**初窥、入门、小成、大成、圆满**

详见：[技能-体技.md](技能-体技.md)

### 2. 灵技

适用范围：

- 主要由炼气体系在 **炼气期及以上** 修炼和使用。
- 依赖灵气修为施展，变化更多，威力通常也更强。

等级结构：

- 品阶：**天、地、玄、黄**
- 熟练度：**初窥、入门、小成、大成、圆满**

详见：[技能-灵技.md](技能-灵技.md)

### 3. 魔法

适用范围：

- 主要由魔法体系修行者修炼和使用。
- 魔法分为 **无、风、雷、水、火、土、冰、光、暗、时间、空间** 等属性。
- **冰属性** 属于水属性的变异属性，存在冰属性魔法，但不对应天然亲和属性。
- 一般人通常只亲和 **一种属性**，但魔法可以通过多个属性进行合成。
- 非亲和属性者也可释放对应属性魔法，只是威力与控制力通常不及亲和属性者。

等级结构：

- 品级：**下级、中级、上级、最上位、圣级、王级、帝级、神级**
- 释放方式：**咏唱、完全咏唱、高速咏唱、二重咏唱、三重咏唱、无咏唱、演奏、魔法阵、手势结印、符文雕刻、组合咏唱、光环被动**

详见：[技能-魔法.md](技能-魔法.md)

### 4. 活气

适用范围：

- 活气以**附加在物件上使用**为核心。
- 活气分为 **大空七属性** 与 **大地七属性**。
- **大空** 为大空七属性之尊，**大地** 为大地七属性之尊。
- **大空** 与 **大地** 互为平等，其余十二属性互为平等。

等级结构：

- 品阶：**一阶、二阶、三阶、四阶、五阶、六阶、七阶**
- 释放方式：**附加在物件上使用**

详见：[技能-活气.md](技能-活气.md)

### 5. 佛法

适用范围：

- 主要由佛魔体系修行者修炼和使用。
- 顺向修行者多以**愿力**施展佛法，逆向修行者多以**业力**施展佛法。
- 施法者可以借用其他修炼者的愿力或业力，但前提是自身**能够承受**。

等级结构：

- 品阶：**本愿法、佛光法、圣法、贤法、凡法**
- 释放方式：**念诵经文**

补充说明：

- 愿力与业力都可以增强佛法的**威力、范围与距离**。
- 佛法的核心施展方式是**念诵经文**，借力施法也必须通过经文完成统合与导出。

详见：[技能-佛法.md](技能-佛法.md)

### 6. 异能

适用范围：

- 主要由改造体系修行者修炼和使用。
- 能量炉会自带一个与自身等级相同的**本命异能**，且**本命异能不可替换**。
- 每次能量炉完成一次大阶段升级，都会新增一个与本命异能相关的异能能力；新增异能可借助 **异能水晶** 替换。

等级结构：

- 等级：**S、A、B、C、D、E、F**
- 成长规则：**具备熟练度；熟练度满后可升级，单个异能最高可升两级**

补充说明：

- 使用异能能力需要消耗**能量**。

详见：[技能-异能.md](技能-异能.md)

## 五、总述

这个世界的修炼逻辑可以概括为：

- **普通人** 是所有人的共同起点。
- **锻体** 是炼气体系、炼体体系与骑士道体系的共同入门阶段，也是接触体技的第一步。
- **骑士侍从** 是骑士道体系在锻体之后的过渡阶段，用于感应未来属性；**见习白骑士**、**见习黑骑士** 与 **见习灰骑士** 则是正式分流起点。灰系在 **灰骑士** 阶段弱于同境白 / 黑系，但进入 **混沌阶段** 后会反超同境 **神圣 / 黑暗系**。
- **魔法学徒** 是魔法体系的起点，意味着开始感知并吸收外界魔力。
- **一阶** 是活气体系的起点，但必须先完成活气觉醒。
- **净信** 与 **染信** 是佛魔体系的起点，分别对应顺向与逆向的入门之路。
- **一级改造者** 是改造体系的起点，但必须先完成能量炉植入并稳定运转一天。
- 若继续走 **灵气修行**，便会进入炼气体系，以灵根和灵技为核心。
- 若继续走 **肉身修行**，便会进入炼体体系，以气血和体技为核心。
- 若继续走 **骑士道修行**，便会在锻体后先经历骑士侍从，再进入白 / 黑 / 灰三支之一，并在后期走向神圣 / 黑暗 / 混沌路线，以誓约、圣暗双属性注技与翼展压制为核心。
- 若继续走 **魔法修行**，便会进入魔法体系，以魔力、术式和精神掌控为核心。
- 若继续走 **活气修行**，便会进入活气体系，以属性觉醒、外物附着和媒介强化为核心。
- 若继续走 **佛魔修行**，便会进入佛魔体系，以心性、愿力、业力与正逆观照为核心。
- 若继续走 **改造修行**，便会进入改造体系，以能量炉、充能进度与异能演化为核心。

七条道路并非完全割裂，但各有偏重：

- 炼气体系强在灵气变化、法术手段与后期飞升。
- 炼体体系强在肉身强度、正面搏杀与气血爆发。
- 骑士道体系强在誓约加持、属性注技、逐阶增长的光翼体系，以及灰系“前期偏弱、混沌后期反超”的路线反转。
- 魔法体系强在魔力操控、术式变化与阶段性飞行能力的逐步提升。
- 活气体系强在属性分系、觉醒潜力与外物附着后的战斗变化。
- 佛魔体系强在心性淬炼、顺逆两道并存，以及以不同心境抵达同一终点的修行逻辑。
- 改造体系强在炉心供能、异能分化与少数变异个体带来的上限跳跃。
