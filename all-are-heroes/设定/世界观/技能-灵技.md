# 灵技

通过驾驭灵气与天地法则，修仙者可以掌握灵技。灵技相比体技更为玄奥，威力更大，需要足够的灵气修为作为基础。

## 品阶等级（高 → 低）

| 品阶 | 说明 |
|------|------|
| 天阶 | 威力最强，难以获得，往往为宗门绝学或上古传承 |
| 地阶 | 威力强大，较为罕见，需要深厚修为才能驾驭 |
| 玄阶 | 威力中等，较为常见，能追踪天地灵脉 |
| 黄阶 | 威力最弱，新手常用，是修仙者入门必备 |

## 修行等级（低 → 高）

| 等级 | 说明 |
|------|------|
| 初窥 | 初次接触灵技，水过地皮湿，仅仅窥了一些皮毛 |
| 入门 | 初步掌握法诀与念诵，需要持续修炼 |
| 小成 | 小有所成，可以稳定使用，灵气流转通畅 |
| 大成 | 深入领悟法则精妙，发挥接近极限，威力大增 |
| 圆满 | 融会贯通，灵气应念而发，可开发新的变化或传授他人 |

## 修炼者

炼气期及以上的修仙者可以掌握灵技，驾驭灵气施展更为强大的法术。

## 组合示例

- **黄阶初窥**：最初级接触，几乎无法应用
- **黄阶入门**：最基础的灵技，适合初修士学习
- **玄阶小成**：威力明显增强，可用于对敌
- **地阶大成**：法力深厚，变化不少
- **天阶圆满**：传说级别的绝学，可翻江倒海

## 与体技、魔法、活气、佛法的区别

- **体技**：依赖肉身气血，战斗方式直接粗暴
- **灵技**：依赖灵气修为，战斗方式玄奥精妙，威力往往更强
- **魔法**：依赖魔力、术式结构与属性亲和，释放方式更多样，元素变化与组合能力更强
- **活气**：依赖活气与外物媒介，不以法诀或咒文为核心，而以属性附着和物件联动见长
- **佛法**：依赖念诵经文与愿力/业力引动，强调心性、借力与经文共鸣，不以灵气法诀为根基
