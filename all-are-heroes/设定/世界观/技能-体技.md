# 体技

通过修炼肉身与筋骨，修行者可以掌握纯粹的体技。体技是炼体修士与骑士道修行者的主要武器，也可作为炼气修士的辅修战斗手段，利用气血、属性与肉身力量施展各种武技。

## 品阶等级（高 → 低）

| 品阶 | 说明 |
| ---- | ---- |
| 天阶 | 威力最强，难以获得，往往为宗门绝学或古老秘传 |
| 地阶 | 威力强大，较为罕见，需要超群的资质才能修成 |
| 玄阶 | 威力中等，较为常见，大多数修士可望学得 |
| 黄阶 | 威力最弱，新手常用，是入门必备 |

## 骑士道称谓（高 → 低）

| 骑士道称谓 | 对应常规品阶 |
| ---- | ---- |
| 大帝级 | 天阶 |
| 国王级 | 地阶 |
| 爵士级 | 玄阶 |
| 凡级 | 黄阶 |

## 修行等级（低 → 高）

| 等级 | 说明 |
| ---- | ---- |
| 初窥 | 初次接触体技，水过地皮湿，仅仅窥了一些皮毛 |
| 入门 | 初步掌握基本动作与要诀，需要持续修炼 |
| 小成 | 小有所成，可以稳定使用，威力初显 |
| 大成 | 深入领悟要义，发挥接近极限，加强变化 |
| 圆满 | 融会贯通，将体技化为本能，可开发新的变化或传授他人 |

## 修炼者

普通人无法使用体技。炼气体系、炼体体系与骑士道体系的修士，自锻体起都可使用体技；这三条道路在**锻体阶段**，体技最高都只能达到**小成**。此外，炼气体系中的修士即便进入炼气、筑基乃至更高境界，体技的修炼上限依旧是**小成**，通常会优先修炼和使用灵技；炼体体系则自锻体之后可以继续精进体技，直至更高层次；骑士道体系则会在 **骑士侍从** 阶段开始感应未来属性，于 **见习骑士** 阶段自行产生对应属性，并在正式成为 **白骑士**、**黑骑士** 或 **灰骑士** 后，从周围吸收同类属性并将其注入体技。灰 / 混沌系始终使用 **圣暗双属性**，只是前期受掣于双属性分流，后期完成统合后反而更强。

## 组合示例

- **黄阶初窥**：最初级接触，几乎无战力
- **黄阶入门**：基础体技，适合初学者
- **玄阶小成**：威力明显增强，可用于对敌
- **地阶大成**：战力显著，威力强大
- **天阶圆满**：武林绝学，威力无穷

## 与灵技、魔法、活气、佛法的区别

- **体技**：普通人之外，炼气、炼体与骑士道三路自锻体起都可使用；三路在锻体阶段都最多只能修到小成，炼气体系整体也最多只能修到小成，炼体体系会继续深修，骑士道则会沿“感应属性 → 自生属性 → 吸收外界属性并注技”的顺序推进，灰 / 混沌系始终以圣暗双属性为本，只是前期偏弱、后期统合后更强
- **灵技**：主要由炼气体系在炼气期及以上修炼和使用，依赖灵气修为，战斗方式玄奥精妙，变化更多，威力往往更强
- **魔法**：主要由魔法体系修行者使用，依赖魔力、术式结构与属性亲和，远中程变化更多，释放方式也更复杂
- **活气**：依赖活气本身与外物媒介，强调属性附着与物件联动，通常并不以纯肉身爆发为核心
- **佛法**：依赖念诵经文以及愿力或业力引动，也可借他人之力增强威力、范围与距离，并不以肉身硬碰硬为核心
