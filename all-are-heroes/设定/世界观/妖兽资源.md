# 妖兽资源

妖兽并不只是灾祸来源，也是灾后时代最重要的超凡材料来源之一。对各国而言，猎杀妖兽从来不是单纯清剿，更是获取药材、兵装材料与突破资源的关键途径。

## 通用认知

- 妖兽自低阶到高阶，资源价值差距极大，越高阶的个体越需要专业队伍分割与封存。
- 同一种妖兽在不同体系修行者手里，利用方式往往并不相同。
- 高阶妖兽尸体长时间暴露可能发生二次异变、毒化或灵性逸散，因此战后回收速度极其重要。

## 常见材料

| 部位 | 常见用途 |
| ---- | ---- |
| 皮、鳞、甲片 | 制作护甲、软甲、盾材、符布与特殊防具 |
| 骨、角、牙、爪 | 打造成刀兵、箭头、枪锋、机关构件与媒介骨架 |
| 血、肉、筋、膜 | 经净化和炮制后入药、入膳，或作为高能补给材料 |
| 毒囊、腺体、目珠 | 制作毒药、淬炼剂、侦测器具与特殊术式材料 |
| 妖丹 | 炼药、炼器、布阵与辅助突破的核心材料 |

## 妖丹规则

妖丹通常位于妖兽心脏要害附近，是妖兽全身能量与异化精华最集中的部分。并非所有低阶妖兽都会稳定结出妖丹，但中高阶妖兽与特殊异种往往都会出现成形妖丹。

- 妖丹可以磨粉入药，是高阶丹药、药浴与恢复药剂的重要材料。
- 妖丹也能作为部分体系突破与升阶时的辅助材料，但只能作为助力，不能替代境界本身需要的积累与领悟。
- 直接吞服未经处理的妖丹风险极高，轻则经脉受损，重则被妖性反噬。
- 不同体系对妖丹的使用方式差异很大，炼气、魔法、佛魔与改造体系都更依赖精炼后的成品，而活气体系更重属性相性与媒介适配。

## 利用原则

- 妖兽资源必须经过净化、分拣、封存与登记，才能进入正规流通体系。
- 基地市通常会设置专门的回收工坊、炼药坊、兵装坊或研究机构处理妖兽材料。
- 高阶妖丹、稀有骨材与特殊腺体往往属于战略物资，会被国家层面统一调配。

## 世界观作用

妖兽资源体系让“清剿妖兽”不只是被动防守，也成为各国维持修行体系、兵装工业与高阶药剂生产的重要基础。正因如此，妖兽潮越猛烈，灾后文明对妖兽材料的依赖也越深。
