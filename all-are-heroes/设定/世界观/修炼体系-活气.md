# 活气体系

以活气觉醒、属性附着与外物承载为核心的修炼道路。活气修行者更依赖媒介与属性分系，通过将活气附着在物件上发挥战力。

## 觉醒方式

活气并非人人天生可用。觉醒方式主要分为 **自行觉醒** 与 **基因药剂觉醒** 两类。其中，**触摸觉醒石觉醒也属于自行觉醒**，因此所有自行觉醒都只能在 **10 岁至 20 岁** 之间完成，而**基因药剂觉醒**只要 **14 岁以后** 便可尝试。

- **触摸觉醒石觉醒**：属于自行觉醒的一种，也是最正统、最常见的自行觉醒方式。
- **自然自行觉醒**：极少数人会在 10 岁至 20 岁期间自然觉醒，超过这一年龄后将无法再通过自然方式完成觉醒。
- **使用基因药剂觉醒**：只要年满 14 岁后即可尝试，以外力刺激身体与血脉，从而强行完成觉醒。

补充规则：

- 通过**自行觉醒**，包括自然自行觉醒与触摸觉醒石觉醒，都有概率觉醒为 **大空** 或 **大地** 属性。
- 通过**基因药剂**觉醒者，只能觉醒 **非尊属性**。
- 通过**基因药剂**觉醒存在失败概率。
- 一个人一生最多只能进行 **三次基因药剂觉醒尝试**。
- 通过基因药剂觉醒者，在后续进阶时只能继续使用**下一阶对应的基因药剂**完成突破。
- 觉醒属性详见：[技能-活气.md](技能-活气.md)

## 觉醒属性

活气觉醒属性分为 **大空七属性** 与 **大地七属性** 两大体系，共 **14 种属性**。

### 大空七属性

| 分类 | 属性 |
| ---- | ---- |
| 尊属性 | 大空 |
| 非尊属性 | 岚、雨、云、晴、雷、雾 |

### 大地七属性

| 分类 | 属性 |
| ---- | ---- |
| 尊属性 | 大地 |
| 非尊属性 | 沼泽、溪流、冰河、森林、山岳、沙漠 |

### 觉醒规则

- **大空** 与 **大地** 为尊属性，地位高于其余属性。
- 通过**自行觉醒**，理论上可觉醒全部 **14 种属性**。
- 正常情况下，一名活气修行者一生只会觉醒 **一种** 活气属性，双属性同体近乎不可能，**大空** 与 **大地** 同体更被视作传说级异常。
- 通过**基因药剂觉醒**，只能觉醒 **12 种非尊属性**，无法觉醒 **大空** 与 **大地**。
- 属性关系与详细设定详见：[技能-活气.md](技能-活气.md)

## 大阶段与小境界

活气体系正式修炼共分 **一阶至九阶** 九个大阶段。

每个大阶段分为 **初期、中期、后期、巅峰** 共四个小境界。

---

### 一、普通人

> 尚未完成活气觉醒的普通人，无境界之分，无法使用活气。

---

### 二、一阶

| 4 | 一阶巅峰 |

## 总览

```text
普通人
  └→ 一阶（初期 ~ 巅峰）【体内产生活气，可使用活气】
       └→ 二阶（初期 ~ 巅峰）【大空/大地可借物飞行】
            └→ 三阶（初期 ~ 巅峰）【其余属性可借物飞行】
                 └→ 四阶（初期 ~ 巅峰）【大空/大地可御空飞行】
                      └→ 五阶（初期 ~ 巅峰）【其余属性可御空飞行】
                           └→ 六阶（初期 ~ 巅峰）
                                └→ 七阶（初期 ~ 巅峰）
                                     └→ 八阶（初期 ~ 巅峰）
                                          └→ 九阶（初期 ~ 巅峰）
```

活气修炼共 **9 大阶段 × 4 小境界 = 36 个修炼境界层次**（普通人无境界划分）。
