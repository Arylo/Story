# 活气

活气是一种依附于外物使用的特殊技能体系。其核心不在于徒手直接释放，而在于将活气附加在物件上，使武器、防具、饰品、机关或其他媒介产生不同属性的强化与变化。

## 属性体系

活气分为两大体系：**大空七属性** 与 **大地七属性**。

| 属性 | 地位 |
| ---- | ---- |
| 大空 | 大空七属性之尊 |
| 岚 | 与其余非尊属性平等 |
| 雨 | 与其余非尊属性平等 |
| 云 | 与其余非尊属性平等 |
| 晴 | 与其余非尊属性平等 |
| 雷 | 与其余非尊属性平等 |
| 雾 | 与其余非尊属性平等 |

### 大地七属性

| 属性 | 地位 |

| 沼泽 | 与其余非尊属性平等 |
| 溪流 | 与其余非尊属性平等 |
| 冰河 | 与其余非尊属性平等 |

| 沙漠 | 与其余非尊属性平等 |

### 属性关系

- **大空** 为大空七属性之尊，其余六属性互为平等。
- **大地** 为大地七属性之尊，其余六属性互为平等。
- **大空** 与 **大地** 互为平等。
- 除 **大空** 与 **大地** 外，其余十二属性互为平等。

可以概括为：

大空 = 大地 > 岚 = 雨 = 云 = 晴 = 雷 = 雾 = 沼泽 = 溪流 = 冰河 = 森林 = 山岳 = 沙漠

## 释放方式

- 活气的核心释放方式是**附加在物件上使用**。
- 可附着的对象包括武器、防具、饰品、弹药、器械、机关以及其他特殊媒介。
- 同一种物件在附着不同活气属性后，会呈现完全不同的效果与战斗风格。

## 使用者

活气使用者通常依赖特定媒介来承载自身属性，通过外物放大活气的特性，而非单纯依赖肉身、灵气或魔力本身。

- 正常情况下，一名活气使用者只会拥有 **一种** 活气属性，双属性同体近乎不可能，**大空** 与 **大地** 同时存在则属于极端异常。

## 与体技、灵技、魔法、佛法的区别

- **活气**：依赖活气本身与外物媒介，强调属性附着、物件强化与媒介联动。
- **体技**：依赖肉身与气血，战斗方式最直接粗暴。
- **灵技**：依赖灵气修为与法诀，偏向术法演化与灵气操控。
- **魔法**：依赖魔力、术式结构与属性亲和，释放方式最复杂。
- **佛法**：依赖念诵经文与愿力/业力引动，可借外来力量增强威力、范围与距离，但不以属性附着和外物承载为核心。
