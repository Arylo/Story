# 佛魔体系

以心性、愿力、业力与执念为核心的修炼道路。修行者入门后会分为 **顺向** 与 **逆向** 两种修炼方向。顺向讲究清净、观照与离欲；逆向讲究逆观、破执与以魔炼心。两路看似相反，最终都通往 **补处** 与 **正觉**。

## 修炼方向

- **顺向**：净信 → 念住 → 明心 → 见道 → 离欲 → 无诤 → 不动 → 补处 → 正觉
- **逆向**：染信 → 逆观 → 破执 → 逆行 → 同体 → 妄语 → 大动 → 补处 → 正觉

## 大阶段与小境界

顺向与逆向都分为 **9 个大阶段**。

每个大阶段分为 **一重至八重** 加 **圆满**，共 **9 个小境界**。

普通人
  ├→ 净信（一重 ~ 八重 ~ 圆满）【可产生愿力】
  │    └→ 念住（一重 ~ 八重 ~ 圆满）【可吸收愿力】

  │                             └→ 不动（一重 ~ 八重 ~ 圆满）【可显现法相】
  │                                  └→ 补处（一重 ~ 八重 ~ 圆满）
  │                                       └→ 正觉（一重 ~ 八重 ~ 圆满）
  └→ 染信（一重 ~ 八重 ~ 圆满）【可产生业力】
    └→ 逆观（一重 ~ 八重 ~ 圆满）【可吸收业力】
         └→ 破执（一重 ~ 八重 ~ 圆满）

                   └→ 同体（一重 ~ 八重 ~ 圆满）【可御空飞行】
                        └→ 妄语（一重 ~ 八重 ~ 圆满）
                             └→ 大动（一重 ~ 八重 ~ 圆满）【可显现法相】

佛魔修炼中，**顺向** 与 **逆向** 各自共 **9 大阶段 × 9 小境界 = 81 个修炼境界层次**（普通人无境界划分）。
