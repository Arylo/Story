# 骑士道体系

以肉身、誓约与圣暗属性淬炼为核心的修炼道路。骑士道以 **锻体** 为基，在 **骑士侍从** 阶段开始感应未来属性，于 **见习白/黑/灰骑士** 阶段自行孕育对应属性，随后沿三脉分别成长为 **白 → 神圣**、**黑 → 黑暗** 与 **灰 → 混沌** 三条独立路线。灰系在 **灰骑士** 阶段会因圣暗双属性彼此分流而弱于同境白 / 黑骑士，但一旦迈入 **混沌阶段** 并完成双属性统合，往往会反过来优于同境 **神圣 / 黑暗系**。

## 骑士分支

- **白 / 神圣系**：以 **圣属性** 为核心，擅长净化、守护、贯穿与正面压制。
- **黑 / 黑暗系**：以 **暗属性** 为核心，擅长侵蚀、吞没、突袭与爆发斩杀。
- **灰 / 混沌系**：始终以 **圣暗双属性** 为核心。前中期因双属性彼此牵制，整体弱于同境白 / 黑系；后期一旦完成统合，双属性联动反而会压过同境神圣 / 黑暗系。

## 大阶段与小境界

骑士道每个大阶段分为 **一重至五重** 加 **圆满** 共六个小境界。

**可使用体技**：骑士道自锻体起即可修炼 **体技**。骑士道内部将体技品级称为 **大帝级、国王级、爵士级、凡级**，实则分别对应常见体技体系中的 **天、地、玄、黄** 四阶。每个品级同样分为 **初窥、入门、小成、大成、圆满** 五个等级。在锻体阶段，体技最高仅能达到 **小成**；**骑士侍从** 阶段已经能感应未来对应属性，但仍**未能**附加在体技上；**见习骑士** 阶段则已可在体内自行产生对应属性；自正式成为 **白 / 黑 / 灰骑士** 起，便可从周围吸收同类属性并将其稳定注入体技之中。

**光翼规则**：自 **骑士** 阶段起，每升一阶都会新增 **一对光翼**。**骑士** 为 **一对**，**精锐骑士** 为 **两对**，**神圣 / 黑暗 / 混沌骑士** 为 **三对**，**领主** 为 **四对**，**帝王** 为 **五对**。其中 **白 / 神圣系** 凝聚 **圣翼**，**黑 / 黑暗系** 凝聚 **暗翼**，**灰骑士** 凝聚 **灰翼**，而 **混沌系** 自混沌骑士起则呈现 **左圣右暗** 的异色翼展。

---

### 一、普通人

> 尚未踏入骑士道之门的普通人，无境界之分，仅以体魄、意志与是否具备骑士誓约资质论高下。

---

### 二、锻体

通过修炼肉身、打磨筋骨与建立誓约根基，为踏入骑士道奠定基础。此阶段修行者开始掌握 **体技**，可以纯粹通过肉身力量进行战斗。在锻体阶段，体技最高仅能达到 **小成**。

| 小境界 | 名称 |
| ---- | ---- |
| 1 | 锻体一重 |
| 2 | 锻体二重 |
| 3 | 锻体三重 |
| 4 | 锻体四重 |
| 5 | 锻体五重 |
| 6 | 锻体圆满 |

---

### 三、骑士侍从

锻体圆满后，修行者进入骑士侍从阶段。此时体内尚不能自行稳定地产生属性之力，却已经能够感应并确认自身更契合的 **圣**、**暗** 或 **圣暗双属性** 倾向，不过仍**未能**将其附加在体技上。

| 小境界 | 名称 |
| ---- | ---- |
| 1 | 骑士侍从一重 |
| 2 | 骑士侍从二重 |
| 3 | 骑士侍从三重 |
| 4 | 骑士侍从四重 |
| 5 | 骑士侍从五重 |
| 6 | 骑士侍从圆满 |

---

### 四、见习骑士

骑士侍从圆满后，修行者会正式分化为 **见习白骑士**、**见习黑骑士** 或 **见习灰骑士**。此阶段已经可以在体内自行产生对应属性，用于蕴养誓印、洗练肉身与稳固未来路线，但仍难像正式骑士那样高效汲取外界同类属性。

| ---- | ---- | ---- | ---- |
| 1 | 见习白骑士一重 | 见习黑骑士一重 | 见习灰骑士一重 |
| 2 | 见习白骑士二重 | 见习黑骑士二重 | 见习灰骑士二重 |
| 3 | 见习白骑士三重 | 见习黑骑士三重 | 见习灰骑士三重 |
| 4 | 见习白骑士四重 | 见习黑骑士四重 | 见习灰骑士四重 |
| 5 | 见习白骑士五重 | 见习黑骑士五重 | 见习灰骑士五重 |
| 6 | 见习白骑士圆满 | 见习黑骑士圆满 | 见习灰骑士圆满 |

### 五、骑士

正式完成受封后，骑士便可从周围吸收对应属性，并将其稳定注入 **体技**。**白骑士** 可注入 **圣属性**，**黑骑士** 可注入 **暗属性**，**灰骑士** 则可同时注入 **圣暗双属性**。不过在这一阶段，灰骑士因为双属性分流与互斥，整体战力通常会**弱于同境白骑士与黑骑士**。同时，此阶段已可凝聚 **一对光翼**，分别为 **圣翼**、**暗翼** 或 **灰翼**，从而进行**短暂御空飞行**。

| 小境界 | 白骑士分支 | 黑骑士分支 | 灰骑士分支 |
| ---- | ---- | ---- | ---- |
| 1 | 白骑士一重 | 黑骑士一重 | 灰骑士一重 |
| 2 | 白骑士二重 | 黑骑士二重 | 灰骑士二重 |
| 3 | 白骑士三重 | 黑骑士三重 | 灰骑士三重 |
| 4 | 白骑士四重 | 黑骑士四重 | 灰骑士四重 |
| 5 | 白骑士五重 | 黑骑士五重 | 灰骑士五重 |
| 6 | 白骑士圆满 | 黑骑士圆满 | 灰骑士圆满 |

### 六、精锐骑士

精锐骑士的誓约更加稳固，属性注入的持续时间、覆盖范围与爆发强度都会显著提升，已能将体技与甲胄、坐骑或兵装进行更深层联动。灰系在此阶段仍未完全解决双属性彼此掣肘的问题，因此**精锐灰骑士**通常依旧弱于同境的**精锐白骑士**与**精锐黑骑士**。此阶段会再增加 **一对光翼**，共具备 **两对光翼**，从而能够**御空飞行**。

| ---- | ---- | ---- | ---- |

| 2 | 精锐白骑士二重 | 精锐黑骑士二重 | 精锐灰骑士二重 |
| 3 | 精锐白骑士三重 | 精锐黑骑士三重 | 精锐灰骑士三重 |

### 七、神圣 / 黑暗 / 混沌骑士

这是骑士道真正的中高层分水岭。**白系** 升华为 **神圣骑士**，**黑系** 升华为 **黑暗骑士**，**灰系** 则将原本并行的圣暗双属性压缩、纠缠并完成高度统合，成为 **混沌骑士**。需要注意的是，混沌骑士**本质上仍旧使用圣暗双属性**，只是两种力量的统御程度远高于灰骑士阶段。也正因如此，混沌骑士自这一阶段开始，往往会**优于同境神圣骑士与黑暗骑士**。此阶段再增 **一对光翼**，共具备 **三对光翼**。其中神圣骑士为 **三对圣翼**，黑暗骑士为 **三对暗翼**，混沌骑士则呈现 **左圣右暗** 的三对异色翼展。

### 八、神圣 / 黑暗 / 混沌领主

领主级骑士已能将誓约压向更大范围战场，属性统御、威慑与指挥能力都更上一层。**神圣领主** 与 **黑暗领主** 继续沿单属性极限推进，**混沌领主** 则仍旧以高度统合的 **圣暗双属性** 统御战场，并通常继续保持对同境神圣 / 黑暗系的优势。此阶段再增 **一对光翼**，共具备 **四对光翼**。神圣领主与黑暗领主分别以四对圣翼、暗翼镇压全场，混沌领主则以四对 **左圣右暗** 的异色翼展制造更强压迫感。

### 九、神圣 / 黑暗 / 混沌帝王

帝王立于骑士道顶端，誓约、肉身与属性近乎完全合一。此阶段会凝聚 **五对光翼**，达到骑士道翼展的极限。**神圣帝王** 与 **黑暗帝王** 各自在单属性道路上走到巅峰，**混沌帝王** 则依旧以最完美的 **圣暗双属性** 统合形态立足顶端，并通常会压过同境神圣 / 黑暗帝王。混沌帝王以五对 **左圣右暗** 的异色翼展维持最危险也最强大的平衡。

## 总览

```text
普通人
  └→ 锻体（一重 ~ 圆满）【可使用体技】
       └→ 骑士侍从（一重 ~ 圆满）【开始感应对应属性，但未能附加在体技上】
            └→ 见习白骑士 / 见习黑骑士 / 见习灰骑士（一重 ~ 圆满）【体内可自行产生对应属性】
                 ├→ 白骑士（一重 ~ 圆满）【可吸收外界圣属性；一对圣翼，可短暂御空飞行】
                 │    └→ 精锐白骑士（一重 ~ 圆满）【两对圣翼，可御空飞行】
                 │         └→ 神圣骑士（一重 ~ 圆满）【三对圣翼】
                 │              └→ 神圣领主（一重 ~ 圆满）【四对圣翼】
                 │                   └→ 神圣帝王（一重 ~ 圆满）【五对圣翼】
                 ├→ 黑骑士（一重 ~ 圆满）【可吸收外界暗属性；一对暗翼，可短暂御空飞行】
                 │    └→ 精锐黑骑士（一重 ~ 圆满）【两对暗翼，可御空飞行】
                 │         └→ 黑暗骑士（一重 ~ 圆满）【三对暗翼】
                 │              └→ 黑暗领主（一重 ~ 圆满）【四对暗翼】
                 │                   └→ 黑暗帝王（一重 ~ 圆满）【五对暗翼】
                 └→ 灰骑士（一重 ~ 圆满）【可吸收外界圣暗双属性；一对灰翼，可短暂御空飞行】
                      └→ 精锐灰骑士（一重 ~ 圆满）【两对灰翼，可御空飞行；弱于同境精锐白/黑骑士】
                           └→ 混沌骑士（一重 ~ 圆满）【三对光翼，左圣右暗；优于同境神圣/黑暗骑士】
                                └→ 混沌领主（一重 ~ 圆满）【四对光翼，左圣右暗；优于同境神圣/黑暗领主】
                                     └→ 混沌帝王（一重 ~ 圆满）【五对光翼，左圣右暗；优于同境神圣/黑暗帝王】
```

骑士道单一路线共 **8 大阶段 × 6 小境界 = 48 个修炼境界层次**（普通人无境界划分）。其中灰系在 **灰骑士** 与 **精锐灰骑士** 阶段通常弱于同境白 / 黑系，但一旦迈入 **混沌阶段**，便会反过来优于同境 **神圣 / 黑暗系**。
