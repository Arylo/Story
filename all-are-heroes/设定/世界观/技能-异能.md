# 异能

异能是由能量炉直接赋予改造者的特殊能力，也是改造体系最核心的战斗与辅助手段。与体技、灵技、魔法、活气或佛法不同，异能不依赖固定招式或经文传承，而更像是与炉心绑定的先天能力。

## 力量来源

- 异能由 **能量炉** 提供驱动力。
- 每次施展异能，都会消耗改造者当前储备的**能量**。
- 异能的发挥，取决于能量炉本身、当前储能以及使用者对异能的掌握程度。

## 等级结构

| 等级 | 地位 |
| ------ | ------ |
| S | 最高等级 |
| A | 次高等级 |
| B | 高等级 |
| C | 中等级 |
| D | 中下等级 |
| E | 较低等级 |
| F | 最低等级 |

## 熟练度

- 异能都具有 **熟练度**。
- 熟练度会随着持续使用、训练与实战不断提升。
- 只有当某个异能的熟练度达到满值后，该异能才具备进一步升级的条件。

## 本命异能与衍生异能

- 能量炉会自带一个与自身等级相同的**本命异能**。
- **本命异能不可替换**，始终与该能量炉直接绑定。
- 每次能量炉完成一次大阶段升级，都会新增一个与本命异能相关的**衍生异能**。
- 衍生异能可以通过 **异能水晶** 进行替换，以调整改造者的战斗与辅助方向。

## 异能升级

- 当熟练度已满时，异能可以继续升级。
- 单个异能在原有基础上，**最高只能提升两级**。
- 异能升级提升的是该能力本身的品质与效果，不等同于能量炉大阶段的晋升。

## 使用者

- 主要由**改造体系**修行者修炼和使用。
- 改造者的核心战斗力，通常由**本命异能**、**衍生异能组合**与**当前能量储备**共同决定。

## 与体技、灵技、魔法、活气、佛法的区别

- **异能**：依赖能量炉与储能驱动，强调本命能力、衍生能力与个体差异化成长。
- **体技**：依赖肉身与气血，战斗方式最直接粗暴。
- **灵技**：依赖灵气修为与法诀，偏向术法演化与灵气操控。
- **魔法**：依赖魔力、术式结构与属性亲和，释放方式最复杂。
- **活气**：依赖活气与外物媒介，强调属性附着、媒介强化与物件联动。
- **佛法**：依赖念诵经文与愿力/业力引动，可借外来力量放大威力、范围与距离。
