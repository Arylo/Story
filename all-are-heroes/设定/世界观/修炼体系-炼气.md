# 炼气体系

## 大阶段与小境界

每个大阶段分为 **一重至九重** 加 **圆满** 共十个小境界。

---

### 一、普通人

> 尚未踏入修炼之门的普通人，无境界之分，仅以体魄强弱论高下。

---

### 二、锻体

通过修炼肉身，强化根骨，为踏入修仙之路奠定基础。此阶段修仙者开始掌握 **体技**，可以纯粹通过肉身力量与修炼进行战斗。在锻体阶段，体技最高仅能达到小成。

**可使用体技**：普通人之外，炼气体系各阶段皆可使用体技；只是自炼气期起，修士通常会优先修炼和使用灵技。需要注意的是，在**整个炼气体系**中，体技的修炼上限均为**小成**。体技分为天、地、玄、黄四个品阶（天阶为尊），每个品阶分初窥、入门、小成、大成、圆满五个等级。

## 总览

```
普通人
  └→ 锻体（一重 ~ 九重 ~ 圆满）【可使用体技】
       └→ 炼气（一重 ~ 九重 ~ 圆满）【驾驭灵气，可使用灵技】
            └→ 筑基（一重 ~ 九重 ~ 圆满）【可借物飞行】
                 └→ 金丹（一重 ~ 九重 ~ 圆满）
                      └→ 元婴（一重 ~ 九重 ~ 圆满）【可御空飞行】
                           └→ 化神（一重 ~ 九重 ~ 圆满）
                                └→ 合体（一重 ~ 九重 ~ 圆满）
                                     └→ 大乘（一重 ~ 九重 ~ 圆满）
                                          └→ 渡劫（一重 ~ 九重 ~ 圆满）
```

炼气修炼共 **9 大阶段 × 10 小境界 = 90 个修炼境界层次**（普通人无境界划分）。

## 灵根体系

修仙者的灵根决定修炼资质与上限。灵根属性决定修炼倾向，灵根数量决定修炼潜力。**单灵根为尊**，拥有单一灵根属性者资质最纯粹。

### 灵根数量分类

| 分类 | 说明 |
|------|------|
| 单灵根 | 尊贵无比，灵气属性纯粹，修炼速度与上限远超他者 |
| 双灵根 | 较为罕见，两种属性相辅相成，各有妙用 |
| 三灵根 | 资质平庸，三种属性杂糅，修炼缓慢 |
| 四灵根 | 资质平凡，属性过杂，难成大器 |
| 杂灵根 | 五种属性，修炼极其困难，几无成仙之望 |

### 灵根属性

| 属性 | 说明 |
|------|------|
| 风 | 速度与锋利见长，身法飘逸，攻势凌厉 |
| 雷 | 爆发力极强，攻击带有麻痹与震慑之效 |
| 水 | 柔中带刚，善于治愈与压制 |
| 火 | 攻击炽烈，破坏力强，克木属 |
| 土 | 防御厚重，耐久持久，克水属 |
| 木 | 生机旺盛，恢复力强，克土属 |
| 阴 | 诡秘幽深，善用暗劲与诅咒，**必为单灵根** |
| 阳 | 光明正大，克制阴属，净化能力突出，**必为单灵根** |

### 灵根组合认知

- **单灵根**（任意属性）：纯粹无暇，修炼一脉绝学可成至强者
- **双灵根**：如水火相融、风雷互补者，各具特色
- **阴灵根** 或 **阳灵根**：必定为单灵根，极其珍稀，往往被视为天人之资

> 灵根数量越少，属性越纯粹，修炼上限越高。拥有单灵根的修仙者，哪怕只是普通属性，也足以君临四方。
